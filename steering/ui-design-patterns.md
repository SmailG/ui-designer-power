# UI Design Patterns

Common UI patterns and best practices for the UI Designer power.

> **Note:** This power uses Gemini 3 Pro Image Preview to generate actual UI mockup images.
> When generating designs, you'll receive both visual mockups and detailed specifications.

## Layout Patterns

### Dashboard Layout
- Sidebar navigation (fixed or collapsible)
- Top header with user actions
- Main content area with cards/widgets
- Responsive grid system (12-column)

### Landing Page
- Hero section with CTA
- Feature sections (alternating layout)
- Social proof / testimonials
- Footer with links

### Form Layout
- Single column for mobile
- Multi-column for desktop
- Progressive disclosure
- Inline validation
- Clear error states

## Component Patterns

### Navigation
- Primary: Top navbar or sidebar
- Secondary: Tabs or pills
- Breadcrumbs for hierarchy
- Mobile: Hamburger menu

### Cards
- Consistent padding (16-24px)
- Clear hierarchy (title, content, actions)
- Hover states for interactive cards
- Shadow for elevation

### Buttons
- Primary: High contrast, single per section
- Secondary: Lower contrast
- Tertiary: Text/ghost buttons
- Sizes: sm (32px), md (40px), lg (48px)

## Spacing Scale

Use consistent spacing scale (8px base):
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
- 3xl: 64px

## Color System

### Semantic Colors
- Primary: Brand color
- Secondary: Complementary
- Success: Green (#10b981)
- Warning: Yellow (#f59e0b)
- Error: Red (#ef4444)
- Info: Blue (#3b82f6)

### Neutral Scale
- 50-900 scale for grays
- Use for text, borders, backgrounds

## Typography Scale

- xs: 12px
- sm: 14px
- base: 16px
- lg: 18px
- xl: 20px
- 2xl: 24px
- 3xl: 30px
- 4xl: 36px

## Accessibility Guidelines

- Minimum contrast ratio: 4.5:1 for text
- Touch targets: Minimum 44x44px
- Focus indicators: Visible and clear
- Semantic HTML elements
- ARIA labels for icons
- Keyboard navigation support
